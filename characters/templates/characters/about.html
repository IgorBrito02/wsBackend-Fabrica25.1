Sobre o Projeto Rick and Morty

Este projeto é uma homenagem ao universo de Rick and Morty, oferecendo informações detalhadas sobre personagens e locais.

Objetivo do Projeto

Criar uma plataforma interativa para fãs explorarem e aprenderem mais sobre o vasto universo de Rick and Morty.

Tecnologias Utilizadas

O projeto utiliza Django para o backend e a API Rick and Morty para obter dados. O frontend é construído com Bootstrap para uma experiência de usuário responsiva e atraente.
